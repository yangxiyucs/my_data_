配置SSH

  • 如果某台电脑需要与Github上的仓库交互，那么就要把这台电脑的SSH公钥添加到这个Github账户上

  • 1.配置SSH公钥入口

  • 2.修改电脑git配置

    • 配置Github注册邮箱和用户名

        vi .gitconfig

        [user]
      email = zhangjiesharp@163.com
      name = zhangxiaochuZXC

  • 3.生成SSH公钥

      ssh-keygen -t rsa -C "zhangjiesharp@163.com"

  • 4.配置SSH公钥

  • 补充：删除旧的秘钥

    • 删除~/.ssh目录，这里存储了旧的密钥

        rm -r .ssh
